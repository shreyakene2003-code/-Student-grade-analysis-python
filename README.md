# -Student-grade-analysis-python
This project analyzes student performance based on various factors such as study time and gender using Python data analysis and visualization techniques.
# 🎓 Student Performance Analysis


## 🎯 Objectives

* Analyze how study time impacts final grades
* Compare academic performance based on gender
* Understand distribution of student grades
* Identify key patterns and trends

---

## 🛠 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 Key Insights

### 📌 1. Gender-Based Performance

* Male students have slightly higher average grades (~10.91)
* Female students average around ~9.97

### 📌 2. Study Time vs Performance

* Weak positive correlation between study time and grades (r ≈ 0.10)
* More study time does not always guarantee higher marks

### 📌 3. Grade Distribution

* Most students score between 8–14
* Average grade is around 10.42
* Some high performers score above 15



## 📌 Conclusion

This project highlights that while study time has some impact on grades, other factors also influence student performance. Data visualization helps uncover meaningful insights for better academic understand
